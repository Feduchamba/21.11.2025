# Простой веб-сервер с проверкой ссылок

## Быстрый старт

1. Скачать либо клонировать репозиторий.
2. Запустить приложение находясь в корне папки `go run cmd/main.go`. Сервис будет доступен по http://localhost:8080

## Хэндлеры

- `POST http://localhost:8080/` - Проверка ссылок и сохранение запроса
  ```json
  {
    "links":["ozon.ru","Yandex.ru","asd"]
  }
  ```

- `POST http://localhost:8080/pastLinks` - Получение pdf по номеру прошлых запросов
  ```json
  {
    "linksNum":[1,2]
  }
  ```

- `POST http://localhost:8080/html` - Html файл со скриптом для pastLinks(для упрощения получения pdf)

## Реализация базы данных

Для реализации сохранения ссылок без внешней зависимости DB использовалась стандартная map. Реализация лежит в файле internal/storage/storage.go
